# 讀書計時花園

這是一個靜態前端專案，提供讀書計時器與花園種植互動功能：

- 登入/註冊機制（帳號不可重複）
- 讀書計時器與植物成長進度
- 草地主頁隨機產生花、草、樹
- 植物成長至 3 小時後自動結算，存入花園
- 花園 8x7 格顯示，並支援每週刷新與結算畫面

## 使用方式

1. 開啟 `index.html`。
2. 註冊新帳號或登入現有帳號。
3. 點選「開始」啟動讀書計時器。
4. 植物會隨讀書時數成長，3 小時後自動收成並種入花園。
5. 點擊左下角花園按鈕進入花園頁面。
